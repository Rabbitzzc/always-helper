## ⭐️ 特性

- 使用 typescript 编写，提供更好的代码提示和类型检查。
- 测试全覆盖，代码可用性得到保证。
- 覆盖 `日期格式化`、`url参数转换`、`浏览器类型判断`、`节流函数`、`字符串/数组处理`、`本地缓存`、`Cookie 增删改查`、`颜色转换`、`DOM 操作`、`随机数生成`、`金额转换` 等前端常用工具函数。


## 📦 开始

### 安装
```sh
# npm 
npm install --save always-lib

# yarn
yarn add always-lib
```

### 使用

```sh
import {} from 'always-lib
```
