## toggleClass

切换元素的 `class`。

> 兼容性：https://caniuse.com/?search=classList

### 参数

`toggleClass(el, className)`

| 参数      | 作用 | 默认值 |
| --------- | ---- | ------ |
| el        | 元素 | 无     |
| className | 类名 | 无     |


### 使用

安装

```sh
yarn add always-helper 
```

使用

```js
import {toggleClass} from 'always-helper'

toggleClass(el, 'className')
```
