## hasClass

判断元素是否存在某个 `Class` 名。

### 参数

`hasClass(el, className)`

| 参数      | 作用                  | 默认值 |
| --------- | --------------------- | ------ |
| el        | 元素                  | 无     |
| className | 需要判断的 Class 类名 | 无     |


### 使用

安装

```sh
yarn add always-helper 
```

使用

```js
import { hasClass } from 'always-helper'

hasClass(el, 'className')
```
